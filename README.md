# Game

This repository contains a tiny demo Ping Pong game implemented with
[Pygame](https://www.pygame.org/).

## Requirements

- Python 3.11+
- `pygame` package

Install dependencies with:

```bash
pip install pygame
```

## Running

Launch the game window using:

```bash
python ping_pong.py
```

Use the `W`/`S` keys for the left paddle and the arrow keys for the right
paddle. The first player to score wins.
